## Tasks Completed

### Task 1: SubtractiveDecoupling module (commit 706ce5d)
Created `project/models/subtractive_decoupling.py` exactly per the locked design:

- Inherits `PointModule`; kwargs `channels=64, hidden=256, norm_layer=nn.LayerNorm, act_layer=nn.GELU`.
- Shared `PointSequential(LayerNorm)`; independent `mlp_marg` and `mlp_seg` (both `PointSequential(MLP(c, hidden, c, act, drop=0))`).
- Zero-initialised `proj_to_semantic = nn.Linear(channels, channels, bias=False)`.
- Per-channel raw `alpha = nn.Parameter(full((channels,), 0.5))` — NO sigmoid.
- Forward:
  1. `shortcut = point.feat` (cached BEFORE LN).
  2. Shared LN, read `x_norm`.
  3. New `Point` built via `addict.Dict` with `coord / grid_coord / batch / offset / feat` + all `serialized_*` keys; `.sparsify()` gives it its own `sparse_conv_feat`.
  4. `mlp_marg` on `point_marg` → `m`.
  5. `m_projected = proj_to_semantic(m)`, `seg_input = x_norm - alpha * m_projected`.
  6. Reuse LN-ed point as `point_seg`; update both `point_seg.feat` AND `point_seg.sparse_conv_feat = sparse_conv_feat.replace_feature(seg_input)` (spconv feature-cache bug avoided).
  7. `mlp_seg` → `s`.
  8. Residual anchor `shortcut` back onto BOTH streams; both `sparse_conv_feat` updated via `replace_feature`.
  9. Attach `point_marg.point_seg = point_seg`; return `point_marg`.
- Self-contained: no import from `decoupled_bfanet_model`; NOT added to `__init__.py`; NOT registered with `MODELS`.

**Smoke check:** instantiation with production kwargs → 70 464 params, `alpha.shape == (64,)`, `proj_to_semantic.weight` all zeros.

- `@MODELS.register_module() class DecoupledBFANetSegmentorV1(nn.Module)`.
- `__init__(num_classes=8, backbone_out_channels=64, backbone=None)` builds:
  - `backbone = build_model(backbone)`
  - `decoupling = SubtractiveDecoupling(64, 256, nn.LayerNorm, nn.GELU)`
  - `seg_head = nn.Linear(64, 8)`, `marg_head = nn.Linear(64, 1)`
  - `ce_loss`, `lovasz_multi(mode="multiclass")`, `margin_bce(reduction="none")`, `margin_lovasz(mode="binary")`
- `_extract_feat` copied verbatim from `semantic_model.py` (unwinds `pooling_parent`/`pooling_inverse`).
- `forward(input_dict)`:
  1. `Point(input_dict)` → backbone → `_extract_feat` → decoupling
  2. `seg_feat = point.point_seg.feat`, `marg_feat = point.feat`
  3. `seg_logits = seg_head(seg_feat)`, `marg_logits = marg_head(marg_feat)`
  4. If `"segment" in input_dict`:
     - `valid_mask = segment != -1`
     - `boundary_mask.float().view(-1)[valid_mask]` and `marg_logits.view(-1)[valid_mask]`
     - `loss = CE + lovasz_multi + margin_bce.mean() + margin_lovasz`
     - Training → `dict(loss=loss)`; eval-with-labels → `dict(loss=loss, seg_logits=...)`
  5. Else → `dict(seg_logits=...)`
