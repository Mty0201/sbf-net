<objective>
Create the redesigned loss class and evaluator that fix the three Phase 9 problems: replace BCE with SmoothL1+Tversky for support supervision, replace the 1+-base focus term with optional Lovasz-on-boundary, and wire both into the project registry.

Purpose: The current SupportGuidedSemanticFocusLoss has a near-saturated BCE support loss (stuck at entropy floor ~0.64) and a focus term that produces 93% redundant CE. This plan replaces both mechanisms with proven alternatives from the support-only baseline.
Output: New loss class, new evaluator, both registered in __init__.py.
</objective>

   ```python

   loss_lovasz = self.lovasz_loss(seg_logits, segment)
   loss_semantic = loss_ce + loss_lovasz
   ```

4. Term 2 — Support SmoothL1 + Tversky per D-01, D-05, D-06:
   ```python
   support_target = support_gt * valid_gt  # continuous regression target
   support_region_gt = (valid_gt > 0.5).float()  # binary Tversky target per D-06
   loss_support_reg = self._weighted_mean(
       self.support_regression_loss(support_prob, support_target), valid_gt)
   loss_support_cover = self._tversky_loss(
       support_prob, support_region_gt,
       alpha=self.support_tversky_alpha, beta=self.support_tversky_beta)
   loss_support = (self.support_reg_weight * loss_support_reg
                   + self.support_cover_weight * loss_support_cover)
   ```

5. Term 3 — Focus (conditional on focus_mode):
   ```python
   if self.focus_mode == "lovasz":
       # D-09, D-10: Lovasz on boundary subset only

       if boundary_mask.sum() > 0:

           segment_boundary = segment[boundary_mask]
           loss_focus = self.boundary_lovasz(seg_logits_boundary, segment_boundary)
       else:
           loss_focus = seg_logits.new_tensor(0.0)
   else:
       loss_focus = seg_logits.new_tensor(0.0)
   ```

6. Total — per D-02 (Variant C: no focus) and D-13 (Variant A: with focus):

   ```python
   total_loss = loss_semantic + loss_support
   if self.focus_mode != "none":
       total_loss = total_loss + self.focus_weight * loss_focus
   ```

   ```python
   return dict(

       loss_lovasz=loss_lovasz,
       loss_support=loss_support,
       loss_support_reg=loss_support_reg,

       loss_focus=loss_focus,
   ```
