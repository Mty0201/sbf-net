  <action>
Create `data_pre/bf_edge_v3/scripts/diagnose_net01.py` that performs the full
stratified diagnosis per D-01, D-02, D-03, D-04. The script must:

**1. Density bucketing (per D-02):**
- Load coord.npy for the scene.
- Load boundary_centers.npz and get the source_point_index array to map each
  boundary center back to a scene point.
- Compute mean kNN distance (k=10) for EVERY scene point using scipy.spatial.cKDTree.
  Query k=11 (includes self), take columns 1:, mean over axis=1.
- For each boundary center, look up the local density = mean_knn_dist[source_point_index[i]].
- Compute percentiles of the boundary-center density distribution:
  P25 = np.percentile(center_densities, 25)
  P75 = np.percentile(center_densities, 75)
- Assign buckets: dense = center_density <= P25, mid = P25 < center_density < P75, sparse = center_density >= P75.
- Also compute the same buckets for scene-level points (for Stage 4 analysis):
  For each scene point, compute scene_density = mean_knn_dist[point_index].
  Use the SAME P25/P75 thresholds (from boundary centers) to bucket scene points.

**2. Stage 2 analysis: cluster survival rate by density bucket:**
- Load local_clusters.npz. The `center_index` array tells which boundary centers
  survived clustering (i.e., were assigned to a valid DBSCAN cluster, not noise).
- For each density bucket (dense/mid/sparse), compute:
  - total_centers: count of boundary centers in that bucket
  - survived_centers: count of boundary centers in that bucket whose index appears in
    local_clusters["center_index"]
  - survival_rate = survived_centers / total_centers
  - noise_rate = 1 - survival_rate
- This directly measures "does DBSCAN eps=0.08 lose more centers in sparse regions?"
- Also compute: mean cluster size per bucket (for surviving centers, what cluster_size
  did they end up in).

**3. Stage 4 analysis: valid yield and weight by density bucket:**
- Load edge.npy: columns are [dir_x, dir_y, dir_z, dist, weight, valid].
  valid = edge[:, 5], weight = edge[:, 4], dist = edge[:, 3].
- For each density bucket of SCENE POINTS:
  - total_points: count of scene points in that bucket
  - valid_points: count where valid == 1.0
  - valid_rate = valid_points / total_points
  - mean_weight: mean of weight for valid points in this bucket
  - high_weight_count: count where valid == 1.0 AND weight >= 0.5
  - high_weight_rate = high_weight_count / total_points
- This measures "does sigma=0.04 decay produce lower valid yield in sparse regions?"

**4. Cross-stage gap attribution:**
- Compute the "coverage gap" for each stage separately:
  - Stage 2 gap = survival_rate(dense) - survival_rate(sparse)
  - Stage 4 gap = valid_rate(dense) - valid_rate(sparse)
  - Stage 4 weight gap = high_weight_rate(dense) - high_weight_rate(sparse)
- The larger gap indicates the primary bottleneck.

**5. Output:**
- Print a human-readable summary table to stdout.
- Write a JSON file to `.planning/workstreams/edge-data-quality-repair/phases/01-net-01-diagnosis/diagnosis_output.json` containing:
