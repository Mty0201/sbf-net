**Test functions:**

1. `test_stage1_boundary_centers_identical(refactored_stage1, reference_dir)`:
   Load `reference_dir / "boundary_centers.npz"`. Assert same keys. For each key, assert `np.array_equal(ref[key], refactored_stage1[key])`. Print field name and shape on failure for diagnostics.

2. `test_stage2_local_clusters_identical(refactored_stage2, reference_dir)`:
   Load `reference_dir / "local_clusters.npz"`. Assert same keys. For each key, `np.array_equal`.

3. `test_stage3_supports_identical(refactored_stage3, reference_dir)`:
   Load `reference_dir / "supports.npz"`. Assert same keys (all 27 fields). For each key, `np.array_equal`. This is the most critical test -- it covers the to_runtime_dict() transition path.

4. `test_stage4_edge_dist_identical(refactored_stage4, reference_dir)`:
   Load `reference_dir / "edge_dist.npy"`. Assert `np.array_equal(ref, refactored_stage4["edge_dist"])`.

5. `test_stage4_edge_dir_identical(refactored_stage4, reference_dir)`:
   Load `reference_dir / "edge_dir.npy"`. Assert `np.array_equal`.

6. `test_stage4_edge_valid_identical(refactored_stage4, reference_dir)`:
   Load `reference_dir / "edge_valid.npy"`. Assert `np.array_equal`.

7. `test_stage4_edge_support_id_identical(refactored_stage4, reference_dir)`:
   Load `reference_dir / "edge_support_id.npy"`. Assert `np.array_equal`.

8. `test_validation_passes_on_refactored_output(refactored_stage1, refactored_stage2, refactored_stage3, refactored_stage4, sample_scene_dir)`:
   Call all 4 validation hooks on refactored output. None should raise. This confirms validation hooks (Plan 02) agree with the refactored pipeline output.

9. `test_inmemory_path_matches_perstage(sample_scene_dir, refactored_stage3)`:
   Run the in-memory path (Stage1 -> Stage2 -> Stage3 in sequence, matching build_support_dataset_v3.py pattern) with default configs. Compare each Stage 3 NPZ field against `refactored_stage3` using `np.array_equal`. This confirms the in-memory bypass path produces identical output after the config migration.
