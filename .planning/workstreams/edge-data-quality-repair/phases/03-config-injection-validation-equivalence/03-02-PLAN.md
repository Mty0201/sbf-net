**validate_boundary_centers(payload: dict) -> None:**
Checks derived from Contract: Stage 1 -> Stage 2 in CROSS_STAGE_CONTRACTS.md:
- Required fields: center_coord, center_normal, center_tangent, semantic_pair, source_point_index, confidence
- M = center_coord.shape[0]; all arrays aligned to M
- center_coord: shape (M, 3), dtype float32
- center_normal: shape (M, 3), dtype float32
- center_tangent: shape (M, 3), dtype float32
- semantic_pair: shape (M, 2), dtype int32; pairs sorted (col 0 <= col 1) when M > 0
- source_point_index: shape (M,), dtype int32
- confidence: shape (M,), dtype float32

**validate_local_clusters(payload: dict, num_boundary_centers: int) -> None:**
Checks derived from Contracts: Stage 2 -> Stage 3, Stage 1 -> Stage 3 read-back:
- Required fields: center_index, cluster_id, semantic_pair, cluster_trigger_flag, cluster_size, cluster_centroid
- K = center_index.shape[0]; C = semantic_pair.shape[0]
- center_index: shape (K,), dtype int32; all values in [0, num_boundary_centers) when K > 0
- cluster_id: shape (K,), dtype int32; all values in [0, C) when K > 0
- semantic_pair: shape (C, 2), dtype int32
- cluster_trigger_flag: shape (C,), dtype uint8
- cluster_size: shape (C,), dtype int32
- cluster_centroid: shape (C, 3), dtype float32

**validate_supports(payload: dict) -> None:**
Checks derived from Contract: Stage 3 -> Stage 4:
- Required fields (Stage 4 minimal read set): support_id, semantic_pair, segment_offset, segment_length, segment_start, segment_end, line_start, line_end
- S = support_id.shape[0]; T = segment_start.shape[0]
- support_id: shape (S,), dtype int32
- semantic_pair: shape (S, 2), dtype int32
- segment_offset: shape (S,), dtype int32
- segment_length: shape (S,), dtype int32
- segment_start: shape (T, 3), dtype float32
- segment_end: shape (T, 3), dtype float32
- line_start: shape (S, 3), dtype float32
- line_end: shape (S, 3), dtype float32
- Bounds check: for each support i, segment_offset[i] + segment_length[i] <= T when segment_length[i] > 0
- Bounds check: segment_offset[i] >= 0

- edge_dir: shape (N, 3), dtype float32
- edge_valid: shape (N,); all values in {0, 1} (or bool-castable uint8/int32)
- edge_support_id: shape (N,), dtype int32
